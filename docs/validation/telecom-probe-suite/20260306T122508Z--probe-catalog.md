# Probe Catalog

Implemented probes:
- `registration_visibility_probe`
- `endpoint_reachability_probe`
- `outbound_trunk_probe`
- `controlled_originate_probe`
- `bridge_formation_probe`
- `cleanup_verification_probe`
- `observability_query_probe`
- `post_change_validation_probe_suite`

Each probe in the registry includes:
- name/title/purpose
- failure-impact class (`risk_class`)
- supported modes and platforms
- fixture/lab support flags
- gating requirements

Execution model per probe:
- `precheck`
- `gating`
- `execute`
- `assert`
- `cleanup`
- `postcheck` (smoke/audit links)
